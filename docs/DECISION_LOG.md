## 决策记录

### 2025-05 - 技术栈选择

**问题**：选择什么技术栈构建 UI 地基

**选项**：
1. React + 纯 CSS
2. React + Tailwind CSS + Radix UI + Motion

**决定**：选项 2

**理由**：
- Tailwind 快速构建样式
- Radix UI 提供无障碍组件基础
- Motion 统一动画系统
- 可参考 shadcn/ui 模式

**影响**：项目将基于这套技术栈构建

---

### 2025-05 - 组件策略

**问题**：如何管理组件

**选项**：
1. 安装 shadcn/ui 完整组件库
2. 手写必要的组件，参考 shadcn/ui 模式

**决定**：选项 2

**理由**：
- shadcn/ui 初始化可能不顺利
- 只实现必要的组件，避免过度
- 手写更好控制和维护

**影响**：按需创建组件，不盲目安装
