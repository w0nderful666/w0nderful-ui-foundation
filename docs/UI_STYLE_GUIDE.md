# UI_STYLE_GUIDE.md - UI 风格指南

## 整体风格

- **现代简洁**：无过多装饰，功能优先
- **Local First 工具感**：像实用小工具的界面
- **Web OS 元素**：可包含 Dock、Window 等类 OS 组件

## 配色方案

### 主色调

- **主色**：`#6366f1` (indigo-500) - 用于主要按钮、强调元素
- **次色**：`#8b5cf6` (violet-500) - 用于次要强调
- **背景**：亮色 `#fafafa`，暗色 `#18181b`
- **文字**：亮色 `#18181b`，暗色 `#fafafa`

### 功能色

- **成功**：`#22c55e` (green-500)
- **警告**：`#f59e0b` (amber-500)
- **错误**：`#ef4444` (red-500)
- **信息**：`#3b82f6` (blue-500)

### 暗色主题适配

所有颜色必须有对应的暗色模式，确保对比度足够。

## 字体

- **主字体**：系统字体 `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- **等宽字体**：用于代码 `"JetBrains Mono", "Fira Code", monospace`
- **字号**：14px 基准，12px 小字，16px 标题

## 间距

- **基准**：4px
- **常用**：8px, 12px, 16px, 24px, 32px
- **组件内**：12px padding
- **组件间**：16px gap

## 圆角

- **小**：4px（按钮、小元素）
- **中**：8px（卡片、输入框）
- **大**：12px（模态框、大卡片）
- **全圆**：9999px（pill 按钮、dock）

## 阴影

- **小**：`0 1px 2px rgba(0,0,0,0.05)`
- **中**：`0 4px 6px -1px rgba(0,0,0,0.1)`
- **大**：`0 10px 15px -3px rgba(0,0,0,0.1)`
- **浮起**：`0 20px 25px -5px rgba(0,0,0,0.1)`

## 动画

- **时长**：150ms（快）、300ms（标准）、500ms（慢）
- **缓动**：ease-out（退出）、ease-in（进入）、ease-in-out（移动）
- **效果**：fade、slide、scale、resize

## 组件风格

### Button

- 高度：32px（默认）、28px（小）、40px（大）
- 圆角：6px
- 过渡：150ms

### Card

- 圆角：8px
- 背景：白色（亮）/ 深灰（暗）
- 内边距：16px

### Dialog

- 居中显示
- 最大宽度：500px
- 圆角：12px
- 遮罩：半透明黑

### Toast

- 位置：右下角
- 圆角：8px
- 自动消失：3-5秒
- 可手动关闭

## 禁止事项

- 不使用过多彩色
- 不使用过度装饰
- 不创建孤立组件
- 不引入第二套风格
